Create User

Step 1

The first step to create user is to make a sign up request:
POST /api/auth/signup

Following object is required when you post the above request:
{
  username: STRING,
  password: STRING,
  email: STRING
}

Remark:
username Cannot be empty and length must be within 1 to 30 characters long
password Cannot be empty and length must be within 1 to 30 characters long

If success, it will return a "Pending User ID" pendingUserId
Then check your email, you will receive a 4-digitcode. You will need the code in next step.

Step 2

The second step is to send a confirm request:
POST /api/auth/signUpConfirm

You need to post the pendingUserId and the 4-digitcode to complete this process:
{
  code: NUMBER,
  pendingUserId: NUMBER,
}

If the request is succeful, a user object will be returned :
user : {
  jwt: STRING,
  user_id: NUMBER,
  user_name: STRING,
  email: STRING,
}

Save the jwt into you local device. You will need this token for identification when you are using the app.Save the new jwt in you device and attach it to every reaquest's (attach in "Authorization"). This token will be valid for 24 hours.

Login

The JWT will be invalid 24 hours after logging in. You will need to log in again in order to get access to different api end points.
To log in, you need to post a request to POST /api/auth/signup with folloing data:
{
  username: STRING,
  password: STRING,
}

If the request is succeful, a user object will be returned :
user : {
  jwt: STRING,
  user_id: NUMBER,
  user_name: STRING,
  email: STRING,
}

Save the new jwt in you device and attach it to every reaquest's (attach in "Authorization");

Post Entry

To Create an entry, you need to post a request to POST /api/entries with folloing data:
{
  intention: STRING,
  entry_body: STRING,
  tarot_card_id: [
    {
       id: STRING,
       isLight: BOOL,
       readingStyle: STRING
     },
        ...
  ],
}

The following is the return from server:
entries : {
  entry_id: NUMBER,
  intention: STRING,
  entry_body: STRING,
  tarot_card_id: [
    {
       id: STRING,
       isLight: BOOL,
       readingStyle: STRING
     },
        ...
  ],
}
